# 🌿 Mahasri Traders – Full Stack E-commerce Website

A responsive e-commerce website developed using HTML, CSS, JavaScript, Firebase Authentication, Cloud Firestore, and Vercel.

## 🚀 Live Website

https://mahasri-traders.vercel.app/

## 📌 Features

- User Login & Registration
- Firebase Authentication
- Product Management
- Shopping Cart
- Wishlist
- Checkout
- Admin Dashboard
- Product Search
- Category Filter
- Price Filter
- Responsive Design
- Vercel Deployment

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript
- Firebase Authentication
- Cloud Firestore
- Git & GitHub
- Vercel
